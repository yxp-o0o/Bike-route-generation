# 智能车竞赛科目二绕八字路径规划工具 🚗
![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-orange)
![License](https://img.shields.io/badge/License-MIT-green)

一款专为**第二十一届全国大学生智能车竞赛单车定向组别科目二绕八字项目**打造的路径规划工具，支持手绘路径、参数化优化、数据复现等核心功能，帮你告别手动算坐标的痛苦(部分功能可能存在问题和Bug，有问题概不负责)

## 核心功能 ✨
### 1. 可视化手绘路径
- 鼠标拖动画出大致路径，自动吸附规划起点，不用手动输坐标
- 实时预览手绘轨迹，松手自动优化，懒人友好

### 2. 全参数可调优化
想怎么调就怎么调，主打一个「参数自由」：
- 锥桶位置/半径自定义（适配不同赛场）
- 步进长度、最大转角、单次转角限制（防翻车）
- 平滑系数、曲率敏感度（让路径丝滑不卡顿）
- 优化迭代次数（卷就完事了）
- 角度模式切换（-180~180° / 0~360° / 累积角度，适配不同惯导）

### 3. 数据复现&验证
- 支持解析C数组/逗号分隔的距离/角度数据，一键复现路径
- 独立复现起点设置，支持从数据中解析XY坐标
- 角度取反、模式自动检测，适配不同硬件输出格式
- 导出PNG截图

### 4. 数据导出&调试
- 打印距离/角度数据到控制台，直接复制到代码里
- 导入CSV数据，批量处理更高效
- 实时显示路径关键点、方向箭头，调试一目了然

### 基础使用流程
1. **设置锥桶参数**：输入锥桶半径、上下锥桶Y坐标，点击「更新锥桶」
2. **配置规划起点**：设置X/Y坐标，点击「设置规划起点」
3. **手绘路径**：鼠标左键拖动画出绕八字轨迹（从起点开始）
4. **参数优化**：调整路径参数（步进、转角、平滑度等）
5. **生成数据**：点击「显示距离/角度数据」，复制到车控代码即可
6. **数据复现**：粘贴已有距离/角度数据，可视化路径

## 核心特性详解 🧠
### 路径优化逻辑
- 自适应重采样：曲率大的地方自动加密点，曲率小的地方稀疏点
- 转角限制：防止单次转角过大导致车辆失控（老司机都懂）
- 锥桶避让：自动避开锥桶碰撞（别问，问就是撞过）
- 高斯平滑：让路径从「锯齿状」变成「丝滑曲线」

### 角度模式适配
| 模式 | 范围 | 适用场景 |
|------|------|----------|
| 模式1 | -180°~180° | 大部分惯导默认输出 |
| 模式2 | 0°~360° | 部分硬件环形角度输出 |
| 模式3 | 累积角度 | 长路径角度累计计算 |

### 数据复现亮点
- 自动检测角度模式，不用手动猜格式
- 独立复现起点，规划/复现互不干扰
- 角度取反功能，解决「路径走反」的坑爹问题
